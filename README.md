Creating an application with Python (Flask) and Typescript/React that will sort out data and return it to the client for the client to render and present to the user

# Requirements
- Node v18
- Python v3

# Running the Front-end
```
cd PreqinTask
npm install
npm run dev
```

# Running the Server
```
cd PreqinTask
npm run start-api
```
